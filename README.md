# Low Power ALU Design Using VLSI

## Overview
A 4-bit Low Power Arithmetic Logic Unit (ALU) implemented using Verilog HDL.

## Operations Supported
- Addition
- Subtraction
- AND
- OR

## Files
- alu.v : ALU Design
- alu_tb.v : Testbench

## Low Power Feature
Enable-based operation control to reduce unnecessary switching activity.

## Tools Used
- Verilog HDL
- EDA Playground
- Vivado
- ModelSim
- Quartus Prime
